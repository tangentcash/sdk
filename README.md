## Project information
The Tangent Blockchain SDK enables developers to perform various cryptographic operations securely and efficiently within the Tangent blockchain ecosystem.
- **ECDSA Authentication & Encryption**
- **Transaction Serialization/Deserialization**
- **Signing & Metadata Extraction**
- **Access to select RPC methods for network interaction**
- **Built-in load balancer and server switching for reliable communication**

This compact toolkit is ideal for developers seeking to integrate with Tangent blockchain.

## Building
Clone this repository
```bash
git clone https://github.com/tangentcash/sdk
```
Install dependencies and build dist
```bash
yarn && yarn make
```
Use as a dependency:
```bash
yarn add tangentsdk@https://github.com/tangentcash/sdk
```
